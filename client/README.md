# Arcadium Client

Web client application for the Arcadium game.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Configuration

Set environment variables:
- `REACT_APP_API_URL` - Game server API URL
- `REACT_APP_WIKI_URL` - Wiki service URL
